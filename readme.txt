== Installation ==

Need help installing WPMinds Growth Blocks? Follow these simple steps:

1. Log in to your WordPress site and navigate to the Plugins → Add New page.
2. In the search bar, type "WPMinds Growth Blocks" and hit Enter.
3. Find our plugin in the list of search results and click on the "Install Now" button.
4. Once installation is complete, click the "Activate" link to start using WPMinds Growth Blocks.

Prefer to install WPMinds Growth Blocks manually? No problem. Just follow these easy steps:

1. Download the WPMinds Growth Blocks plugin from WordPress.org.
2. Unzip the downloaded package and move the unzipped folder to your WordPress plugin's directory.
3. Log in to your WordPress site and navigate to the Plugins screen.
4. Locate WPMinds Growth Blocks in the list of plugins and click on the "Activate" link.

Yes, you can use WPMinds Growth Blocks with any theme which have the customized option, but we recommend using one of the with the WP Minds Growth theme.

Yes, the FAQ block has a schema option.
